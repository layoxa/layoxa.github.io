Emergclinic (Toronto)

Emergclinic connects patients to nearby dental clinics for prompt and dependable care during dental emergencies. Our platform offers instant access to trusted professionals, ensuring help is just a click away.

Key features include:

  • Responsive web application for both providers and patients
  • Notification system for SMS and email
  • Nearby search capability based on patient location
  • Credit system for clinics to manage and charge credits for specific actions
  • Secure payment processing with Stripe integration, including card storage
  • HIPAA-aligned database to ensure patient data security (Leyoxa designs its data infrastructure to align with HIPAA/PHIPA technical safeguard requirements.)
  • CI/CD pipeline for efficient development and deployment
  • Multiple staging environments for development and production

Smile Link (Texas)

Smile Link is the first dedicated patient app designed specifically for dental offices. It streamlines patient access, appointment scheduling, and communication, while enabling dentists to foster improved patient engagement and operational efficiency.

Key features include:

  • Custom integration with their dental practice management system (OpenDental)
  • Real-time, end-to-end encrypted chat system
  • Native mobile apps for Android and iOS
  • Dashboard with role-based access
  • Customizable notification system using user-defined templates via SMS, Email, or Push Notifications
  • Two-way messaging system for seamless communication between dental clinics and patients via SMS
  • Data warehouse supporting multiple integrations, including OpenDental, Dentrix, and more
  • Booking system for multiple providers and operatories in dental clinics
  • HIPAA-aligned database to ensure patient data security (Leyoxa designs its data infrastructure to align with HIPAA/PHIPA technical safeguard requirements.)
  • CI/CD pipeline for efficient development and deployment
  • Multiple staging environments for development and production
