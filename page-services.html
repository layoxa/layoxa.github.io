Emergclinic (Toronto)

Emergclinic connected patients to nearby dental clinics for prompt and dependable care during dental emergencies. The site is no longer live, but the founder walk-through demo on YouTube shows how it worked end to end.

Key features included:

  • Responsive web application for both providers and patients
  • SMS & email notification system
  • Nearby search based on patient location
  • Credit system for clinics to manage and charge for specific actions
  • Secure payment processing via Stripe with card-on-file
  • HIPAA-aligned database (Leyoxa designs its data infrastructure to align with HIPAA/PHIPA technical safeguard requirements.)
  • Business Associate Agreements (BAAs) available for covered entities upon request
  • CI/CD pipeline + multiple staging environments

SmileLink (Texas)

SmileLink is a dedicated patient app and clinic dashboard designed specifically for dental offices. It streamlines patient access, appointment scheduling, and communication while giving dentists a clear path to better engagement.

Key features include:

  • Custom integration with OpenDental practice management
  • Real-time, end-to-end encrypted chat
  • Native iOS and Android mobile apps
  • Role-based access dashboard
  • Customizable templates for SMS, email, and push notifications
  • Two-way SMS messaging between clinic and patient
  • Data warehouse with multiple PMS integrations (OpenDental, Dentrix)
  • Booking system for multiple providers and operatories
  • HIPAA-aligned database (Leyoxa designs its data infrastructure to align with HIPAA/PHIPA technical safeguard requirements.)
  • Business Associate Agreements (BAAs) available for covered entities upon request
  • CI/CD pipeline + multiple staging environments
